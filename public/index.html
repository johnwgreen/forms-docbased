Environment Variables

Ensure you have the following environment variables set in .env file 
                    
                        AEM_URL='' #AEM Instance
                        AEM_USER_NAME='' # AEM Username
                        AEM_PASSWORD='' # AEM Password
                        AEM_TOKEN='' #Either AEM_TOKEN or AEM_USER_NAME and AEM_PASSWORD is mandatory for fetching content from AEM
                        EDS_URL='' #EDS Endpoint

Render Form present in EDS

Prefix the EDS URL with /doc i.e. /doc/path/to/hlx/page

Render AEM Form

Prefix /aem followed by Adaptive form path without /content/forms/af
